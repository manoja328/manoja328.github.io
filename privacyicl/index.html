Privacy Preserving In-Context-Learning Framework for LLMs

Read Paper View GitHub

Overview

The Privacy-Preserving In-Context Learning (PP-ICL) framework addresses the critical challenge of using Large Language Models (LLMs) with sensitive data. Unlike traditional methods that require computationally expensive fine-tuning (like DP-SGD), PP-ICL generates high-utility synthetic data directly through private inference.

Our method utilizes a private prediction aggregation strategy that provides strong Differential Privacy (DP) guarantees while maintaining the semantic quality of the generated text, making it suitable for downstream training and analysis.

Key Advantages

Zero Fine-Tuning

Achieve privacy without modifying model weights or re-training, significantly reducing GPU costs.

DP Guarantees

Worst-case theoretical privacy bounds ensure that individual data points remain protected.

High Utility

Outperforms existing baseline methods on benchmark tasks including classification and text generation.
